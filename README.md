# Personal Projects Dashboard

A minimalist landing page aggregating links to personal projects hosted on Vercel.

Design inspired by [Hermes Agent](https://hermes-agent.nousresearch.com) — dark theme, industrial minimalism, warm midground tones.

**Live at:** [personal-projects.vercel.app](https://personal-projects.vercel.app) _(soon)_

## Projects

- **[Expense Tracker](https://expense-tracker-liart-gamma.vercel.app)** — Personal finance dashboard with portfolio tracking and analytics
- **[Workout Logbook](https://workout-logbook.vercel.app)** — Weekly training planner with set logging and progression charts
- **[MTG Cube Tracker](https://mtg-cube-tracker.vercel.app)** — Draft session logger with Swiss tournaments and card stats

## Stack

- Static HTML + CSS
- Space Grotesk + JetBrains Mono (Google Fonts)
- Hosted on Vercel
